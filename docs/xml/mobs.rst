モンスター制御
==============
デフォルトの状態では全てのモンスター、動物のスポーンがブロックされています。<mobs>ノードを使用する事でモンスターのスポーンをカスタマイズする事が出来ます。

.. code-block:: xml

   <!-- ゲーム開始10秒経過後ゾンビのみスポーン許可 -->
   <mobs>
       <filter>
           <all>
               <mob>zombie</mob>
               <elapsed min="10s"/>
           </all>
       </filter>
   </mobs>
